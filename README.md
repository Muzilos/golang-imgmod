# muzage

## Usage
```bash
go build main.go
go run ./main --input <input_file_name> --output <output_file_name>
```

### Arguments
``` bash
  --input -i string
        jpeg format image file to read (default "./img/painting.jpg")
  --output -o string
        jpeg format filename for output (default "./img/modded.jpg")
```
